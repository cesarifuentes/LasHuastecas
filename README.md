## 🛠️ Installation & Start Up

1. Run MongoDB service.

  ```
  brew services start mongodb-community
  ```

2. Start frontend. 
  ```
  cd las_huastecas
  npm start
  ```

3. Start server. 
  ```
  cd server
  npm install express body-parser cors mongoose nodemon
  node index.js || npx nodemon index.js
  ```
